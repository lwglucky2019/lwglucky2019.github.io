git创建工程

发表于 2019-09-29 | 分类于 编程

创建新项目

  1. 在github上创建新项目;
  2. 打开git bash;
  3. 进入工作目录；
  4. git clone 下来
  5. 或者 git init, git add . (加入所有文件) / 或者 git add (xx你要加入的文件)
  6. git config —gobal user.name XXXX ; git config —gobal user.email xxx@xxxx
  7. 对于 clone 下来的项目，使用 git remote set-url origin [url] git 的服务器项目，或者 git remote rm origingit , remote add origin [url] , git remote add origin https://github.com/XXXXXX/MathInput.git
  8. git commit -m “加入注释说明”
  9. git push 或者（git push -u origin master）
  10. git pull https://github.com/youraccount/yourproject.git (把项目的git配置拉下来)

如果 git clone总是失败， git config —global http.postBuffer 524288000 设置幻从去大小
