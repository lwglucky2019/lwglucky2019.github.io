OpenCV在树莓派上的安装

发表于 2019-10-13 | 分类于 树莓派

安装必要的库

更新源

sudo apt-get update
sudo apt-get upgrade
sudo rpi-update
sudo reboot

安装库

sudo apt-get -y install build-essential git cmake pkg-config
sudo apt-get -y install libjpeg8-dev
sudo apt-get -y install libtiff5-dev
sudo apt-get -y install libjasper-dev
sudo apt-get -y install libpng12-dev
sudo apt-get -y install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev
sudo apt-get -y install libgtk2.0-dev
sudo apt-get -y install libsdl2-dev
sudo apt-get -y install libatlas-base-dev gfortran
sudo apt-get -y install libxvidcore-dev libx264-dev

获取 opencv代码

git clone https://github.com/opencv/opencv.git
git clone https://github.com/opencv/opencv_contrib.git
如果总是下载到一半就中断，加入 —depth=1参数,比如：
git clone https://github.com/opencv/opencv.git —depth=1

将 opencv_contrib 目录拷贝到 opencv目录


创建 build目录，并 cd build

编译

cmake -D CMAKE_BUILD_TYPE=Release -D CMAKE_INSTALL_PREFIX=/usr/local ..
没有错误就可以编译，如果有错误，就一一修正。有可能需要安装 ffmpeg支持 mpeg 视频流。 ffmpeg可以github上下载安装。
然后
make -j4
sudo make install
sudo ldconfig

测试

python
import cv2
